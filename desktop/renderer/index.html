                Files from this repository help improve suggestions.

                Off by default. When on, copy your prompt (Ctrl+C) from a coding app, like Cursor, VS Code, or a terminal, and a small popup shows a focused rewrite to paste back. It ignores copies from other apps, reads only your clipboard locally, and never records your screen.

                Terminal agent capture is a separate, off-by-default mechanism: it only sees text typed inside a session you explicitly start with metriq-wrap claude or metriq-wrap codex — nothing outside that terminal session, no screen reading, no OS permission required. See the desktop README for setup.

                Editor capture reads only the currently-focused text field in Cursor/VS Code via the macOS accessibility API (not a screenshot or screen recording), and only while that app is frontmost. Inserting a rewrite brings the app to the foreground and replaces the field's contents — Metriq re-checks the field is unchanged immediately before writing, but on a genuine timing race it's possible for an insert to land somewhere unintended. Off by default; available on macOS only.

                AI-tailored rewrite sends your draft prompt directly to Anthropic's API using the key you provide — never through Metriq's own servers. The key is encrypted at rest via your OS keychain, the same guarantee as your login session, and is never displayed again after saving. Off by default, and always falls back to the offline rewrite with no network call if disabled, unset, or unreachable.
